# rp1 Homebrew Tap

Official Homebrew tap for the rp1 CLI.

## Installation

```bash
brew install rp1-run/tap/rp1
```

## Updating

```bash
brew upgrade rp1
```

## About

rp1 provides AI-assisted development workflows for Claude Code and OpenCode.

- **Website:** https://rp1.run
- **GitHub:** https://github.com/rp1-run/rp1
